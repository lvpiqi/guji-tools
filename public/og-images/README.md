# OG Images 说明

Open Graph 图片用于社交媒体分享时的预览图。

## 规格要求

- **尺寸**: 1200 x 630 像素
- **格式**: PNG 或 JPG（推荐 PNG）
- **文件大小**: 建议小于 300KB

## 文件命名

- `default.png` - 默认通用图片
- `{tool-id}.png` - 工具专用图片，如 `remove-finger.png`

## 生成方法

1. 使用设计工具（Figma、Canva 等）创建 1200x630 的图片
2. 包含工具名称、简短描述、品牌 Logo
3. 导出为 PNG 格式
4. 放置到此目录
